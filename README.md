# 📚 Livraria SQL Database

Projeto de banco de dados de uma livraria desenvolvido em SQL.

## 📌 Sobre o projeto
Este projeto tem como objetivo a criação de um banco de dados relacional para uma livraria, incluindo:
- criação do esquema do banco de dados
- inserção de dados para testes
- consultas SQL para análise das informações

## 🗂️ Estrutura do projeto
- `schema_livraria.sql` → criação das tabelas
- `insert_data.sql` → inserção dos dados
- `queries.sql` → consultas SQL para análise

## 🛠️ Tecnologias utilizadas
- MySQL
- SQL

## 📊 Exemplos de consultas
O projeto inclui consultas como:
- listagem de clientes e livros
- filtragem de dados com WHERE
- cálculos com atributos derivados
- ordenação com ORDER BY
- agrupamento com GROUP BY e HAVING
- junções entre tabelas (JOIN)

## 🎯 Objetivo
Praticar conceitos de modelagem de dados e consultas SQL, desenvolvendo um projeto simples de banco de dados.
